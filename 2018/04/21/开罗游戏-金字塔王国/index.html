开罗游戏-金字塔王国

发表于 2018-04-21 | 分类于 娱乐 | | 阅读次数

整个游戏追求：钱、顾客好感度、金字塔力（形状与大小）

建造

这部游戏确实比较有特色，除了之前都市大亨里的城市建筑设施里去掉了电子相关，然后加入了埃及特色建筑
由于金字塔本身楼梯，石头就有讲究，店铺之间则没有城市里那种特色街区的设定，会变复杂

  1. 金字塔特色-基础：楼梯、石头、压顶石、飞毯场、入口、绿植、尖碑、喷泉、狮身人面像、壁画、石像面像
  2. 金字塔特色-用：不同档次房间、浴场、舞蹈、游乐沙场、蛇店、骆驼、宝石、按摩、玻璃工厂
  3. 普通建筑-吃：食堂、咖喱、面包、蔬菜、肉店、牛店、鱼店、餐具、养鸡、绿茶红茶
  4. 普通建筑-用：锤子、书店、洗衣、杂货、鞋店、纺织品、学校、香水

任务

建造任务

  1. 建造金字塔本身的形状也有要求，每个月会竞赛形状好的奖励多

委托任务

贸易任务

  1. 需要携带自己的工艺品，所以需要有人去研究工艺品

人物

  1. 指导调整状态
  2. 人物有擅长的工作
  3. 每个员工可以改变策略：建造、采掘石头、接纳顾客、购物、工艺品、自动

经营

就是提升自身的一些物品道具与系统能力触发，需要消耗一定资源

  1. 可以提高全体员工的素质
  2. 可以提高建筑的等级，翻新工程
  3. 恢复员工体力
  4. 可以出售道具
  5. 可以驱赶眼镜蛇
  6. 可以加入高级的协会
